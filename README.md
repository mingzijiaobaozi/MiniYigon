## 功能介绍 
    
义工小程序, 为进一步掀义工服务活动高潮，助力文明城市创建，近日新增“义工”小程序，不仅为各单位、各部门义工服务展示交流搭建了有效平台。功能包括平台资讯，爱心榜，志愿者风采，品牌项目，志愿培训，志愿活动报名等，采用腾讯提供的小程序云开发解决方案，无须服务器和域名
